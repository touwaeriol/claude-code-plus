# Claude Code Plus Swing UI V2 - 使用指南

## 🚀 快速开始

### 启动

```bash
.\gradlew jetbrains-plugin:runIde
```

### 首次使用

1. 打开 IDEA 右侧工具窗口 → "Claude Code Plus"
2. 输入消息，按 Enter 发送
3. 观察实时打字效果和工具调用

---

## ✨ 核心功能

### 1. 实时对话

- **发送**: Enter
- **换行**: Shift + Enter
- **实时效果**: 看到 Claude 逐字回复
- **Markdown**: 自动渲染代码块、表格、列表

### 2. 工具可视化

每个工具都有专门的 UI：
- **Read**: 点击文件路径打开
- **Edit**: 点击查看 Diff
- **Bash**: 显示命令和输出
- **Grep**: 显示搜索结果

### 3. 上下文管理

- 点击"📎 添加上下文"
- 选择文件或文件夹
- 标签显示在输入框上方
- 点击 × 删除

### 4. 模型切换

底部工具栏：
- 默认 / Sonnet / Opus / Haiku / Opus Plan
- 动态切换，下次对话生效

### 5. Token 监控

**两个位置**：
1. 底部工具栏：📊 总计
2. 生成时浮动栏：实时 in/out/total
